vscode_color_theme
====

Usage
---

`.vscode\extensions` の直下に shka-dark または shka-light フォルダを丸ごと置く。

`Ctrl+Shift+p` で color とでも入力して Preferences: color theme からテーマを設定する。

※ VSCODEマケプレでの公開は諸事情につきやらない。
